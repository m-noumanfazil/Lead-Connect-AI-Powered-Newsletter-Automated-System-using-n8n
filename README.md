#  Lead Connect – AI Powered Personalized Newsletter Automated System

## Why This Exists
Lead Connect automates personalized tech newsletters for subscribers based on their interests and preferred reading tone. Instead of sending generic emails, it dynamically generates content, images, and email copy tailored to each subscriber.  

It exists because most email campaigns are generic, boring, and fail to engage. StudyMate fixes that with automation + AI personalization.

## What Problem It Solves
- Sends curated tech news to the right audience.
- Saves hours of manual newsletter creation.
- Personalizes content to each subscriber’s interests and tone preferences.
- Automates lead collection and newsletter delivery.

## Features
- **Lead Capture Form**: Collects subscriber info (name, email, interest, tone).  
- **Dynamic Content Generation**: Fetches relevant news articles and creates AI-generated newsletters.  
- **AI-Powered Personalization**: Adjusts tone, greeting, and content based on subscriber’s interest.  
- **Image Integration**: Uses Unsplash API to include relevant images automatically.  
- **Automated Email Sending**: Fully automated Gmail integration for sending personalized emails.  
- **n8n Workflow**: Entire automation runs via n8n, from lead capture to newsletter dispatch.

## Tech Stack
- **n8n** – Workflow automation  
- **Google Gemini API** – AI newsletter summarization and tone adjustment  
- **Google Sheets** – Subscriber data storage  
- **Unsplash API** – Image fetching  
- **Gmail API** – Sending emails  
- **Python** – Code nodes in workflow
